
# QuickChat

**Short Description:**

This application is an online chat where users can send text messages. The app displays a list of all sent messages, showing the author, the message text, and the time of sending. Users can use a form to enter data, which includes a field for the authorâ€™s name (optional) and a text field for the message (required), as well as a button to send the message.
